# docker-factory

Centralized Docker images factory, deployed on GHCR.

## Usage

```bash
docker pull ghcr.io/albr21/<image_name>:<image_tag>
```

## Available Images

- [Python 3.13 Alpine](./docs/python-3.13-alpine.md)
- [Trivy](./docs/trivy.md)

## Quick Deployment

All images are deployed using [GitHub Actions](https://github.com/albr21/docker-factory/actions).
